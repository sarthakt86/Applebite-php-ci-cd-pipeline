# Automated CI/CD Pipeline for Modular PHP Application
**Stack:** Jenkins | Docker | Ansible | PHP

## Overview
Implemented a CI/CD pipeline for a modular PHP application with multiple environments.  
Integrated Jenkins for automation and Docker for containerized builds.

## Key Highlights
- Automated build → test → deploy pipeline using Jenkins  
- Dockerized PHP environments  
- Provisioned servers using Ansible playbooks  
- Deployed across dev, staging, and production environments  

## Outcome
⚡ Faster delivery with reliable, repeatable deployments  
✅ Fully automated CI/CD workflow with zero manual intervention
